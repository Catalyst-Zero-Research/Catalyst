# Catalyst Data

The public repository intentionally does not include the processed Materials
Project snapshot. The live demo is backed by a hosted `v2025.09.25` processed
snapshot on the Catalyst demo server.

Expected local data location when running the backend yourself:

```text
data/processed/catalyst/v2025.09.25/
```

Runtime state is generated under `data/local/` and is not committed, except for
non-secret examples and agent contract templates.
